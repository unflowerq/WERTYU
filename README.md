# WERTYU

WERTYU 키보드를 이용해 키보드를 친다 - WERTYU 키보드는 참조링크
타이핑 도중 양손이 자판 기준으로 한키씩 오른쪽으로 이동 되었다.

오타가 났다.
'Q' 인데, 'W' 를 입력했고, 'O' 인데, 'P'를 입력한 것이다. 

주어진 메세지가 있을 때, 이를 보정하라.

예외 상황
- '`', 'Q', 'A', 'Z' 의 경우, 정상 입력으로 간주한다.
- Tab, BcakSpace, Space, Control 키 의 경우, 정상 입력으로 간주한다.

input example
- O S, GOMR YPFSU/

output example
- I AM FINE TODAY.
